# Axion

Axion is a Windows desktop application built with **WPF (.NET)** focused
on structured time tracking and directory-based organization. The
application allows users to manage timers, browse hierarchical entries,
and persist data in a human-readable format.

The project emphasizes: - MVVM architecture - Strong separation of UI
and logic - Reusable styling and theming (Light / Dark mode) -
Predictable state handling and bindings
